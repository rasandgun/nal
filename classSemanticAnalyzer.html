Detailed Description

Класс, занимающийся семантическим анализом

Содержит методы, позволяющие проводить семантический анализ. В процессе анализа поддерживает области видимости, проверяет совместимости типов, проверяет находится ли continue или break внутри цикла, наличие return у non-void функции, и т. д.

Процесс анализа

  1. Сбор всех объявлений функций и глобальных переменных
  2. Обход AST с проверкой семантических правил
  3. Построение таблицы символов с учётом областей видимости

Выполняемые проверки

  • Объявление переменных перед использованием
  • Совместимость типов в операциях и присваиваниях
  • Корректность вызовов функций (количество и типы аргументов)
  • Наличие return в не-void функциях
  • Правильность использования break/continue (только внутри циклов)
  • Типы условий в if, while, for (должны быть скалярными)
