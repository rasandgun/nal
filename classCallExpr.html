Detailed Description

Узел AST, соответствующий вызову функции

Представляет вызов функции с аргументами. Пример: foo(), bar(1, 2), printf("Hello")

std::vector<Expression*> args;
args.push_back(new LiteralExpr(LIT_INT, "42", line, col));
CallExpr* call = new CallExpr("print", args, line, col);
